## 🚀 How to Use

### To View Pages:
1. Open `home.html` in browser (default homepage)
2. Click navigation links to visit other pages
3. All links work locally and will work online

### To Customize:
1. Edit text content directly in HTML files
2. Change colors in `shared.css` (look for `:root` CSS variables)
3. Update contact info in footer and contact page
4. Add blog posts in `blog.html`

### To Deploy:
1. Upload all `.html` and `.css` files to hosting
2. Set `home.html` as the default index page
3. Or rename `home.html` to `index.html` for some hosts
